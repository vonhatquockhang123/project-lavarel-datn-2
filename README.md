## Getting started

```TN
- Clear cache
   php artisan config:clear
    php artisan cache:clear
    php artisan route:clear
    php artisan view:clear
    php artisan clear-compiled
    php artisan optimize
    composer dump-autoload


php artisan route:clear
php artisan config:clear
php artisan cache:clear
php artisan view:clear



- Run
   composer update
   php artisan key:generate
   php artisan migrate
   php artisan db:seed
   php artisan serve.
```
