## Getting started

```TN
- Clear cache
    php artisan config:cache
    php artisan cache:clear
    php artisan config:clear

- Run
   $ composer update
   $ php artisan key:generate
   $ php artisan migrate
   $ php artisan db:seed
   $ php artisan serve.
```
